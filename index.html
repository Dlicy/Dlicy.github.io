fmt包

fmt 是 Go 语言标准库中的一个包，用于格式化输入和输出。它提供了一组函数和类型，用于在控制台、文件和其他输出目标上进行格式化的输入和输出操作。

以下是 fmt 包提供的一些常用功能：

  1. 格式化输出：fmt.Print、fmt.Println、fmt.Printf 等函数用于在控制台或标准输出上打印输出。它们可接受不同类型的参数，并根据需要进行格式化。例如，fmt.Printf 允许使用格式字符串来指定输出的格式和占位符。

  2. 格式化输入：fmt.Scan、fmt.Scanf、fmt.Scanln 等函数用于从输入源（如标准输入）读取输入，并将其解析为指定的类型。这些函数可以根据格式字符串或其他规则进行输入解析和分割。

  3. 字符串拼接：fmt.Sprintf 函数用于将格式化的数据写入字符串，并返回生成的字符串。它可以将不同类型的值格式化并连接成一个字符串。

  4. 错误输出：fmt.Errorf 函数用于创建一个新的错误对象，并将错误消息格式化为字符串。它常用于在处理错误时生成自定义的错误消息。

  5. 自定义输出格式：fmt 包还提供了一些接口，如 fmt.Stringer 和 fmt.Formatter，用于自定义类型的输出格式化。

除了上述功能之外，fmt 包还提供了其他一些函数和类型，用于处理各种格式化和打印输出的需求。它是 Go 语言中常用的包之一，用于快速进行简单的输入和输出操作。

这只是 fmt 包的一小部分功能，它还提供了更多的功能和选项，可以根据需要进行使用和探索。

fmt输出函数分析

1.fmt.Println:
用于打印一行，并在最后自动添加一个换行符 (\n)。
可以接受多个参数，并在它们之间插入空格。
fmt.Println(“Hello”, “World”)

2.fmt.Printf:
类似于C语言的 printf 函数，使用格式化字符串来控制输出格式。
支持格式化占位符，如 %s、%d 等。
不会自动添加换行符，你需要手动添加 \n 或者使用 %v 格式化占位符输出一个换行符。
name := “John”
age := 30
fmt.Printf(“Name: %s, Age: %d\n”, name, age)
// 输出: Name: John, Age: 30

3.fmt.Print:
用于打印一行，但不会自动添加换行符。
可以接受多个参数，并在它们之间插入空格。
fmt.Print(“Hello”, “World”)
// 输出: HelloWorld

fmt.Printf 使用格式化字符串，不自动添加换行符。

标题写法

标题按大小分为6级分别为

title

title

title

title

title
title

使用方法

使用#来写，不同的#个数表示标题的大小。一个#为1级标题，最大。6个为6级标题，最小。
例”# title”

可选语法

Heading level 1

写法：

“Heading level 1”
“===============”

Heading level 2

写法：

“Heading level 2”
“———————-“

注意：

“#”与标题之间最好有一个空格。
标题行上下最好留出一个空白行。

段落

分段方式有两种，一种是在要分段的两端文字中间加上空白行
就像这样：
段落1

段落2；

“段落1”

“段落2”

第二种是在上一段的结尾加上两个空格(好像用不了)
段落1
段落2

设置字体

使用与_相同效果且可以混用 size 斜体 写法：”“+内容+”“
size 加粗 写法：” “+内容+” “
size 删除线 写法：” ~~ “+内容+” ~~ “
**size 加粗斜体 写法：” “+内容+” * “

分割线

使用三个（及以上）的”*”,”-“,”_” 。

第一种写法

使用4个空格或者Tab键缩进
