markdown语法

Posted on 2023-11-20 Edited on 2023-11-21

标题写法

标题按大小分为6级分别为

title

title

title

title

title
title

使用方法

使用#来写，不同的#个数表示标题的大小。一个#为1级标题，最大。6个为6级标题，最小。
例”# title”

可选语法

Heading level 1

写法：

“Heading level 1”
“===============”

Heading level 2

写法：

“Heading level 2”
“———————-“

注意：

“#”与标题之间最好有一个空格。
标题行上下最好留出一个空白行。

段落

分段方式有两种，一种是在要分段的两端文字中间加上空白行
就像这样：
段落1

段落2；

“段落1”

“段落2”

第二种是在上一段的结尾加上两个空格(好像用不了)
段落1
段落2

设置字体

使用与_相同效果且可以混用 size 斜体 写法：”“+内容+”“
size 加粗 写法：” “+内容+” “
size 删除线 写法：” ~~ “+内容+” ~~ “
**size 加粗斜体 写法：” “+内容+” * “

分割线

使用三个（及以上）的”*”,”-“,”_” 。

代码块

第一种写法

使用4个空格或者Tab键缩进

#include<bits/stdc++.h>
using namespace std;
int main(){
    return 0;
}

第二种写法

前后各使用一个”`”符号包裹

`#include

using namespace std;  
int main(){  
    return 0;  
}`

第三种写法

前后各使用3个”`”符号包裹

使用c++则在前3个”`”后+”cpp”

引用

we
are

we

写法： 在被引用的文本前加”>”符号以及一个空格。

引用的嵌套使用

we

are

aw

写法： 使用多个”>”符号来表示各级引用。
注： 引用中也可以写标题等

列表

无序列表

  • 无序列表1
  • 无序列表2
    写法：使用”*”,”+”,”-“并加上一个空格。后续跟上列表名

有序列表

  1. 有序列表1
  2. 有序列表2
    写法：使用数字和一个”.”并加上一个空格来表示

混合使用

    • list1
    • list2
    1. list1
    1. list2
    1. list1
    1. list2
注：列表可以和其他（如引用等）混合使用

表格

学号 姓名 分数
小明 男 75
小红 女 80

写法：
学号|姓名|分数

-|-|-

小明|男|75

小红|女|80

对齐方式：

使用”:”,”:”在左边则左对齐，在右边则右对齐；两边都有则居中对齐；

学号 姓名 分数
小明 男 75
小红 女 80
写法：

学号|姓名|分数

-|:-|-:

小明|男|75

小红|女|80

特殊符号

\\ * _ \{\} # + - . !

以上字符可以通过前加反斜杠”\”来解决

<链接地址>
